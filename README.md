# Old database parser(Education and Personnel)
### Build Application:
``dotnet build``
### Run Application in NET6:
``dotnet run ``
### Run Application in Docker:
``docker-compose up --build -d ``
### Install Version «.NET»
[.NET6](https://github.com/dotnet/core/blob/main/release-notes/6.0/6.0.2/6.0.2.md)
